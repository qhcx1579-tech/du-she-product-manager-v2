# 产物路径

默认所有交接文档写入 `artifacts/`。

建议路径：

- `artifacts/SPEC.md`
- `artifacts/BRIEF.md`
- `artifacts/DESIGN.md`
- `artifacts/PLAN.md`
- `artifacts/BUILD_LOG.md`
- `artifacts/REVIEW.md`
- `artifacts/BUGFIX.md`
- `artifacts/GOAL.md`
- `artifacts/RUN_STATE.md`
- `artifacts/RELEASE.md`

如果项目还没有 `artifacts/` 目录，执行任何主线技能前先创建它。

命名规则：

- 稳定交接文档使用全大写英文文件名。
- 临时草稿可以放在 `artifacts/drafts/`。
- 不上传的内部文件记录在 `artifacts/PRIVATE_EXCLUDE.md`。
