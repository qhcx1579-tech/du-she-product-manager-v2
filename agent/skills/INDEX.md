# Skill Index

## Main Pipeline

- `product-spec-builder`: 需求收集，把模糊想法写成 `artifacts/SPEC.md`
- `design-brief-builder`: 设计规范，把感觉词写成 `artifacts/BRIEF.md`
- `design-maker`: 设计图和高保真方案，写成 `artifacts/DESIGN.md`
- `dev-planner`: 开发计划，写成 `artifacts/PLAN.md`
- `dev-builder`: 按计划开发，写入 `artifacts/BUILD_LOG.md`
- `release-builder`: 隐私审计、打包发布，写入 `artifacts/RELEASE.md`

## Support Skills

- `orchestrator`: 调度层，控制运行节奏、防止死循环和 429
- `bug-fixer`: 修 BUG，写入 `artifacts/BUGFIX.md`
- `skill-maker`: 创建或更新 skill
- `goal-writer`: 写目标，写入 `artifacts/GOAL.md`
- `evolution-steward`: 管自进化，维护 `evolution/EVOLUTION_LOG.md`
- `independent-reviewer`: 独立审核，写入 `artifacts/REVIEW.md`
